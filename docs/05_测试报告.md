# 测试报告

## 测试范围
- 登录/注册
- 活动列表/详情/筛选
- 活动创建/编辑/提交审核/下架/归档
- 报名/候补/取消报名
- 签到码生成/签到
- 通知中心
- 汇总统计/活动统计
- 黑名单管理

## 测试环境
- 前端：Vue 3 + Element Plus
- 后端：Spring Boot 3.5.9
- 数据库：MySQL 8.x
- 端口：前端 8080，后端 8081

## 测试方式
- 手工测试为主
- 未执行自动化测试（可补充）

## 用例摘要
1. 登录成功后按角色跳转正确
2. 组织者创建活动 → 提交审核 → 管理员审核通过 → 学生可报名
3. 报名满员后进入候补，取消后候补转正
4. 签到码生成后学生可签到
5. 通知广播发送后可在通知中心查看
6. 黑名单添加后用户无法报名
