# 部署说明

## 前置条件
- JDK 21
- Maven 3.9+
- Node.js 16+（Vue CLI）
- MySQL 8.x

## 数据库初始化
1. 创建数据库并授权用户
2. 默认启用 `schema.sql` + `data.sql` 初始化
   - 配置：`backend/src/main/resources/application.properties`
   - `spring.sql.init.mode=always`（生产建议改为 `never`）

## 后端启动
```bash
cd backend
./mvnw spring-boot:run
```
默认端口：`8081`

## 前端启动
```bash
cd frontend
npm install
npm run serve
```
默认端口：`8080`

## 前后端联调
前端通过 `frontend/vue.config.js` 代理 `/api` 到 `http://localhost:8081`。

## 生产部署建议
- 前端：`npm run build`，产物部署到 Nginx
- 后端：`mvnw clean package`，运行 `java -jar target/*.jar`
- 关闭初始化脚本，使用生产数据库
```properties
spring.sql.init.mode=never
```
